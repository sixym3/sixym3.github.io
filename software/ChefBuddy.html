ChefBuddy

An AI-powered recipe manager inspired by Paprika V3

Problem

Existing recipe management apps, like Paprika v3, often misclassify web-scraped information and lack AI-powered features for meal planning and pantry tracking. Users must manually adjust recipes, plan meals, and track ingredients, making meal prep tedious.

Solution

ChefBuddy enhances recipe management with AI-driven automation. Built using Express.js, Vue.js, OpenAI, Cheerio, and Axios, it offers:

  • Accurate recipe scraping & summarization using AI to extract key details.
  • AI-powered meal planning to suggest and adjust meals based on user preferences.
  • Pantry tracking with receipt scanning, leveraging OCR and AI to update inventory.
  • Automated ingredient tracking, reminding users to confirm missing or mismarked items.

The app maintains a structured, future-proof JSON schema with versioning to prevent data loss and ensure flexibility for future updates.

What I Learned

  • Designing a scalable JSON schema for AI communication.
  • Frontend-backend interactions and planning for future integrations.
  • Using web scraping tools effectively for structured data extraction.
  • The importance of planning ahead to ensure modular and scalable design.

Challenges Faced

  • Structuring a flexible JSON schema that supports AI interactions and future expansions.
  • Ensuring accurate web scraping while handling inconsistencies in extracted data.

Future Plans

  • Multi-user authentication for shared meal planning.
  • Hosting a small-scale test deployment to gather user feedback.
  • Mobile support for on-the-go accessibility.
  • Voice recognition integration for hands-free recipe management.
