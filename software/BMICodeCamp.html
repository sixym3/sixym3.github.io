Biomedical ML Bootcamp

Completed Fall 2023

Biomedical Data Analytics & Machine Learning Bootcamp - Stony Brook Medicine & CEAS Department intensive program

Program Overview

This intensive bootcamp was provided by Stony Brook Medicine and the Stony Brook CEAS Department in Fall 2023, focusing on machine learning techniques specifically applied to biomedical data analytics and model optimization.

The program provided hands-on experience with fundamental ML concepts, emphasizing both supervised and unsupervised learning methods with real-world biomedical applications and datasets.

Learning Challenge

Understanding and implementing machine learning techniques is essential for effective data analysis and model optimization in biomedical research. The challenge was to master various ML approaches and apply them to real biomedical datasets with clinical relevance.
