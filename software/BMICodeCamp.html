Biomedical Data Analytics & Machine Learning Bootcamp

This Bootcamp was provided by Stony Brook Medicine and the Stony Brook CEAS Department in Fall of 2023.

Problem

Problem

Understanding and implementing machine learning techniques is essential for effective data analysis and model optimization. This bootcamp provided hands-on experience with fundamental ML concepts, focusing on supervised and unsupervised learning methods.

Solution

Throughout the bootcamp, I worked with the Iris dataset and applied various ML techniques:

  • K-Means clustering: Used for unsupervised classification, evaluating metrics such as silhouette scores.
  • Feature selection using SelectKBest: Improved model efficiency by selecting the most relevant features.
  • Random Forest and Decision Trees: Applied for classification tasks.
  • Neural Network Development: Built and optimized models using PyTorch, fine-tuning layer depth and node count.
  • Edge Detection with OpenCV: Processed images using custom convolution kernels.
  • Hyperparameter Tuning with Optuna: Optimized model performance for better accuracy.

What I Learned

  • Gained a solid foundation in ML techniques and evaluation metrics.
  • Understood the importance of hyperparameter tuning for performance improvement.
  • Developed proficiency with Python libraries like Scikit-learn, Pandas, Seaborn, PyTorch, and OpenCV.

Challenges Faced

  • Fine-tuning neural network architectures to achieve optimal learning.
  • Selecting the right hyperparameters efficiently, which was addressed using Optuna.

Future Goals

I now seek opportunities to gain hands-on experience by applying these techniques to real-world datasets and industry applications.
