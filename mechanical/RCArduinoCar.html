RC Arduino Car

Completed Fall 2021

MEC 101: Freshman Design Innovation - Bluetooth-controlled car with custom four-bar steering mechanism

Project Summary

This project involved the design and construction of a Bluetooth-controlled car using an Arduino microcontroller. The primary goal was to create a wirelessly controlled vehicle capable of forward, reverse, left, and right movement.

This project distinguished itself by incorporating a custom-designed four-bar mechanism for steering the front wheels, rather than relying solely on differential speed control. The result was a functional and responsive remote-controlled car with proper Ackermann steering geometry.

Custom Four-Bar Steering Mechanism

Technical Challenge

This project aimed to explore the potential of Bluetooth technology for a more modern and versatile control interface. Many simpler RC cars use differential speed (varying the speed of the left and right wheels) for steering, which can be imprecise and less stable.

🎯 Primary Goals

  • Implement Bluetooth wireless control interface
  • Design mechanically accurate steering system
  • Create custom four-bar linkage mechanism
  • Achieve Ackermann steering geometry

⚡ Technical Challenges

  • Interface Bluetooth module with Arduino effectively
  • Process control signals and translate to motor movements
  • Control servo motor actuating steering linkage
  • Design and fabricate four-bar mechanism

Key Learnings

Microcontroller Programming

Gained proficiency in programming the Arduino using the Arduino IDE and C++, learning to interface multiple sensors and actuators simultaneously.

Wireless Communication

Learned how to interface a Bluetooth module with Arduino and implement wireless data transmission/reception for real-time control applications.

Motor Control Systems

Developed skills in using motor drivers for DC motor control (speed and direction) and servo motor control for precise angular positioning in steering applications.

Mechanical Design

Gained experience designing and building a four-bar steering mechanism, including considerations for geometry, material selection, and Ackermann steering principles.

Problem-Solving

Encountered and overcame technical challenges including debugging wiring issues, refining control logic, and ensuring proper functioning of the four-bar linkage system.

Project Impact

This project successfully demonstrated the integration of Bluetooth communication with both motor and steering control, resulting in a functional remote-controlled car with proper Ackermann steering geometry. It provided hands-on experience with microcontroller programming, wireless communication, and mechanical design principles.
