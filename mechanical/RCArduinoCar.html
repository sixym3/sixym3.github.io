RC Arduino Car

Summary:

This project involved the design and construction of a Bluetooth-controlled car using an Arduino microcontroller. The primary goal was to create a wirelessly controlled vehicle capable of forward, reverse, left, and right movement. This project distinguished itself by incorporating a custom-designed and built four-bar mechanism for steering the front wheels, rather than relying solely on differential speed control. The project successfully demonstrated the integration of Bluetooth communication with both motor and steering control, resulting in a functional and responsive remote-controlled car with Ackermann steering geometry.

Problem:

This project aimed to explore the potential of Bluetooth technology for a more modern and versatile control interface. Furthermore, many simpler RC cars use differential speed (varying the speed of the left and right wheels) for steering, which can be imprecise and less stable. This project sought to implement a more mechanically sound and accurate steering system. The challenge lay in effectively interfacing the Bluetooth module with the Arduino, processing the control signals, translating them into precise motor movements and controlling the servo motor actuating the four-bar steering linkage. This included the design and fabrication of the four-bar mechanism itself.

Materials Used:

  • Arduino Uno
  • HC-05 Bluetooth Module
  • L298N Motor Driver
  • 9V Battery
  • DC Motors with wheels
  • Servo Motor
  • Laser cut plastic parts for the frame
  • Jumper Wires and Breadboard
  • Arduino IDE
  • A Smartphone with ArduinoBlue

What I Learned:

  • Microcontroller Programming: Gained proficiency in programming the Arduino using the Arduino IDE and C++.
  • Bluetooth Communication: Learned how to interface a Bluetooth module with the Arduino and transmit/receive data wirelessly.
  • Motor Control: Developed skills in using a motor driver to control DC motors, including speed and direction.
  • Servo Motor Control: Learned how to control a servo motor for precise angular positioning.
  • Hardware Interfacing: Improved my understanding of how to connect various electronic components together.
  • Mechanical Design: Gained experience designing and building a four-bar steering mechanism, including considerations for geometry and material selection.
  • Problem-Solving: Encountered and overcame several technical challenges, including debugging wiring issues, refining the control logic, and ensuring the proper functioning of the four-bar linkage.

Looking Ahead:

  • Implementing more advanced features like obstacle avoidance using ultrasonic sensors.
  • Designing a more robust chassis for the car.
  • Exploring the use of different types of motors or drive systems for improved performance.
  • Adding feedback mechanisms, such as encoders, to provide more precise control over the car's movements.
  • Custom print the rack and pinion for the four-bar mechanism for improved steering performance and durability.
