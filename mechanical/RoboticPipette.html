Project Challenge

Industrial robots excel at movement but lack the fine liquid handling needed for scientific research. Existing systems provide milliliter accuracy, insufficient for protocols requiring microliter precision.

Design Requirements

  • 50 microliter accuracy for scientific protocols
  • Quick swap capability for multiple pipette volumes
  • Integration with existing tool exchanger system
  • Use standard laboratory pipettes without modification
  • Internal cabling compatibility with other laboratory grippers
  • ROS2 integration for automated research scheduling systems

Close-up: 3D Printed Integration

Mechanical Engineering Solution

Quick Swap Mechanism

Enables rapid pipette changes between different volumes without interrupting automated workflows.

  • Secure locking mechanism - Reliable engagement during operation
  • Universal compatibility - Works with standard laboratory pipettes
  • Repeatable positioning - Consistent alignment for accuracy

Dual Actuator System

Two independent 3D printed actuators replicate human pipetting actions with microliter precision and dual control modes.

  • Primary actuator - Main plunger control with position feedback
  • Secondary actuator - Tip ejection mechanism with force limiting
  • Linear precision - Stepper motor control for repeatable actuation
  • Integrated mounting - Combined with quick swap holder design

Technical Challenges & Solutions

Quick Swap System Requirements

Quick swap capability drove the entire system architecture. Laboratory workflows require rapid switching between pipette volumes, but the UR5e tool exchanger prohibited external wires, forcing integration through the robot's internal 8-pin RS485 interface.

  • Researched internal wiring requirements and RS485 protocol
  • Implemented Arduino-based communication
  • Validated compatibility with existing laboratory grippers

Precision & Integration

Achieved 5µL accuracy through position-controlled linear actuators with force feedback, while integrating multiple software layers from low-level Arduino control to high-level ROS2 research scheduling systems.
