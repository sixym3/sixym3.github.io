Project Challenge

Industrial robots like the UR5e excel at precision movement but lack the fine liquid handling capabilities required for scientific research. Existing robotic dispensing systems are designed for industrial applications (gluing, coating) with accuracies in the milliliter range - insufficient for scientific protocols requiring microliter precision.

Design Requirements

  • 50 microliter accuracy for scientific protocols
  • Integration with existing UR5e robotic arm
  • Use standard laboratory pipettes
  • Remote control capability for automated workflows
  • ROS2 integration for research scheduling systems

Mechanical Engineering Solution

3D Printed Holder Design

Designed and 3D printed a precision holder to securely mount standard laboratory pipettes to the UR5e end effector. The custom design ensures proper alignment and prevents movement during operation while allowing easy pipette replacement.

  • Quick-release mechanism for pipette swapping
  • Rigid mounting to eliminate flex during actuation
  • Universal compatibility with standard pipette sizes
  • Lightweight PLA 3D printed construction

3D Printed Dual Actuator System

Designed and 3D printed two independent actuators to replicate human pipetting actions with microliter precision. The custom actuator system provides both coarse and fine control for optimal accuracy across different volume ranges.

  • Primary 3D printed actuator for main plunger control
  • Secondary 3D printed actuator for tip ejection mechanism
  • Stepper motor precision for repeatable actuation
  • Force feedback to prevent over-compression

Technical Challenges & Solutions

Internal Cabling & Communication Protocol

The UR5e robot features a tool exchanger system that allows automatic swapping between different end effectors and grippers. This meant external wires running down the robot arm were not feasible - the pipette system had to integrate through the robot's internal cabling using the standard 8-pin quick connect/disconnect interface (compatible with other robotic grippers in the lab).

Technical Implementation Process

  • Researched robot's internal wiring requirements and pin configurations
  • Learned RS485 industrial communication protocol for reliable data transmission
  • Conducted pin mapping and connectivity testing for the 8-pin interface
  • Integrated RS485 shield with Arduino microcontroller system
  • Configured specialized shielding to prevent electromagnetic interference
  • Validated compatibility with existing robotic grippers in laboratory

This self-directed investigation and implementation enabled seamless tool swapping while maintaining robust communication between the robot controller and pipette system, ensuring compatibility with the laboratory's existing multi-gripper workflow.

Precision Actuation Requirements

Achieving 50μL accuracy required precise control of the pipette plunger mechanism while maintaining compatibility with standard laboratory pipettes of various sizes.

Mechanical Challenges

  • Consistent force application
  • Universal pipette compatibility

Control Challenges
